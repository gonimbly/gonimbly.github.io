Features

Authentication

  • Login with username and password, stored in a postgres database
  • Signup with fullname, email, and password

Questions

  • List of quantifiable questions
  • Swipe to reveal rating stars
  • Tap star to submit answer
  • Dismiss questions
  • View previously answered / dismissed questions

Rewards

  • Wallet of points
  • Carosel of redeemable rewards
  • Confirmation when redeeming reward
  • Animation when reward is redeemed
